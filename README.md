== Installation ==
Installing MB WP Custom Login & Registration is just like any other WordPress plugin. Navigate to your WordPress “Plugins” folder:
Follow the instructions:
Download the zip version from here and paste it inside your plugin directory and unzip it, so easy. 
Then visit /wp-admin of your project and active the plugin.

Then create two pages. The first page named \"user-login\"  and the second page named \"user-register\".  
Then paste the shortcodes [MB_login_form] to the login page and [MB_register_form] to the register page.
Done.

SHORTCODES
[MB_login_form] and [MB_register_form]
